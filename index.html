Pig Dice

JavaScript version of the classic dice game "Pig Dice".

Rules: Roll until you want to hold. When you hold, the dice are passed and the round score is added to your total score. If you hit a one, the dice pass automatically and you lose that round score. First one to 100 wins.

This project furthered my knowledge of objects, how to store and change key values within them, and behavior driven development. The github can be found here. Enjoy!
